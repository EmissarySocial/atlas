# Atlas

![A map of the world from the Age of Discovery](https://github.com/EmissarySocial/atlas/blob/main/meta/world-map.png)

Atlas is a social mapping server (built on [Emissary](https://emissary.dev)) that lets you post **geo-located notes** on the **Fediverse**.  

Please see the [Project Wiki](https://github.com/EmissarySocial/atlas/wiki) for a detailed setup guide, and more information about the project.
